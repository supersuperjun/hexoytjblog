hexo-admin 后端

2024/6/20

已折旧
这个项目已经贬值了，我不再使用它了

六合彩
一个 hexo 的管理插件，支持 hexo v3.x，使用
gulp + babel + webpack 和
es6 + angular +material + jwt构建

演示
http://nihgwu.github.io/hexo-hey/
用户名：hexo
密码：hey

截图
桌面 移动的

关于
客户端框架：angular
仪表板 UI：材料
身份验证：jwt
markdown 解析器：标记
markdown 样式：github-markdown-css
Markdown 编辑器：codemirror
用法

  1. 安装 hexo-hey
    npm install hexo-hey –save
  2. 添加管理配置部分到_config.yml：

Admin

admin:
name: hexo
password: hey
secret: hey hexo
expire: 60*1
# cors: http://localhost:3000
如果你想在其他地方（例如 CDN）为你的客户提供服务，cors 是可选的

  1. 服务 Hexo
    hexo serve
    然后访问http://localhost:4000/admin，并使用上一步设置的账户登录
